# 0to1 DevHub MVP Roadmap

## Phase 1: Foundation
- Create workspace, env examples, local GSD artifacts, package scripts.

## Phase 2: Backend API
- Implement Express API, strict validation, rate limiting, Supabase insert, SQL schema.

## Phase 3: Frontend Experience
- Implement authorized 0to1.wtf-style Next.js landing page, waitlist form, Lenis, Anime.js, smoke canvas hero, responsive states.

## Phase 4: Verification
- Run builds/type checks, API checks, browser screenshots, and security review.

## Phase 5: Clone Polish
- Applied Stitch UI design guidance, refined fonts from Syne/Space Grotesk to Montserrat/Inter, and tuned hero smoke/typography from screenshots.
